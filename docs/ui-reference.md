# UI Reference

Reference source: `/web-to-colon/main.html`

Homepage shell:
- Compact top navbar with left brand, centered category links, and right-side utility actions
- Search area directly below the navbar with a primary search field, quick search chips, and a small trending list panel
- Filter bar beneath search using quick chips plus labeled select-style controls
- Dense media poster grid with card bodies below each poster
- Simple static pagination shell at the bottom for layout fidelity only

Poster card:
- 2:3 ratio
- top-left status badge
- top-right rating badge
- bottom-right availability badge
- title
- year and format line
- small availability stats row

Responsive targets:
- Mobile: stacked search and filter panels, 2-column poster grid
- Tablet: 3 to 4 columns depending on width
- Desktop: up to 6 poster columns

Visual direction:
- Dark theme background with subtle gradient layers
- Clean bordered surfaces instead of a large hero section
- Layout fidelity prioritized over real data or backend behavior
